函数

使用function关键字声明函数，

function outname(name:string):void{
    console.log('你好'+name+"!")
}

//第二种写法：
let outname = (name:string)=>{
    console.log('你好'+name+"!")
}

outname('lucy')

ts还提供了一种可传参数和不传参数也执行的函数的操作，写法如下：

let outname = (name:string='张三')=>{
    console.log('你好'+name+"!")
}
