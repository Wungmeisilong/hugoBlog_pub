网站内常用到的模版

代码分制表

以< tabs title="file name" >，< /tabs >结尾； 然后里面写以% tab title="code name" style="info" color="blue " %，% /tab %结尾 他们都需要用{{}}括起来

​
{{< tabs title="hello." >}}
//代码块一
{{% tab title="py" %}}
    ```markdown
    **这里写代码**
    ```
{{% /tab %}}
//代码块二
{{% tab title="sh" %}}
    ```markdown
    **这里写代码**
    ```
{{% /tab %}}

{{< /tabs >}}
